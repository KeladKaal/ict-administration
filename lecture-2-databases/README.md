# Лекция 2 — Базы данных

В базе данных живут данные приложения. devops следит, чтобы она была доступна, сохранна и быстра.

**Блок 1 — Основы эксплуатации БД.** PostgreSQL/MySQL: установка, пользователи и права, подключения, connection pooling.

**Блок 2 — Надёжность.** Репликация (primary/replica), бэкапы и восстановление, миграции схемы.

**Блок 3 — Эксплуатация и проблемы.** Мониторинг, медленные запросы, типичные аварии. Кратко про NoSQL — когда реляционка не подходит.
