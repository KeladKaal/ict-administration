# Lecture 2 — Databases

The database is where the application's data lives. devops makes sure it's available, safe, and fast.

**Block 1 — Database basics.** PostgreSQL/MySQL: installation, users and permissions, connections, connection pooling.

**Block 2 — Reliability.** Replication (primary/replica), backups and restore, schema migrations.

**Block 3 — Operations and problems.** Monitoring, slow queries, common incidents. A short look at NoSQL — when a relational database isn't the right fit.
