# Lecture 5 — Storage, delivery and security

Where files live, how they reach users fast, and how to keep all of it locked down.

**Block 1 — Object storage.** S3: buckets, access policies, lifecycle rules, presigned URLs.

**Block 2 — Content delivery.** CDN (caching static content), nginx/reverse proxy, load balancing.

**Block 3 — Security (cross-cutting).** TLS, managing secrets (Vault), and access control across all the systems in this course.
