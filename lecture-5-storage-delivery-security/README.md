# Лекция 5 — Хранилище, доставка и безопасность

Где лежат файлы, как они быстро доезжают до пользователей и как всё это закрыть от посторонних.

**Блок 1 — Объектное хранилище.** S3: бакеты, политики доступа, lifecycle-правила, presigned URLs.

**Блок 2 — Доставка контента.** CDN (кэширование статики), nginx/reverse proxy, балансировка нагрузки.

**Блок 3 — Безопасность (сквозной блок).** TLS, управление секретами (Vault) и контроль доступа ко всем системам из курса.
