# Лекция 7 — AI-инфраструктура

Новый набор инструментов, который devops настраивает для команд: модели, агенты и данные за ними.

**Блок 1 — Модели и доступ.** Облачные API vs self-hosted модели (Ollama/vLLM), ключи/лимиты/расходы, прокси перед LLM.

**Блок 2 — Автоматизация и агенты.** n8n (self-hosted workflow), агенты, tool calling, MCP, skills/toolkit.

**Блок 3 — Данные и эксплуатация.** Векторные БД (Qdrant/pgvector), RAG, а также наблюдаемость и безопасность AI-систем.
