# Lecture 7 — AI infrastructure

The new set of tools devops sets up for teams: models, agents, and the data behind them.

**Block 1 — Models and access.** Cloud APIs vs self-hosted models (Ollama/vLLM), keys/limits/cost, putting a proxy in front of an LLM.

**Block 2 — Automation and agents.** n8n (self-hosted workflows), agents, tool calling, MCP, skills/toolkits.

**Block 3 — Data and operations.** Vector databases (Qdrant/pgvector), RAG, plus observability and security for AI systems.
